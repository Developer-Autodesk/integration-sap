##Description

*This sample is part of the [Developer-Autodesk/Autodesk-View-and-Data-API-Samples](../) repository.*

An ASP.Net MVC website sample that integrates the Autodesk View and Data API with an SAP database using the [SAP Gateway for Microsoft (GWM)](http://scn.sap.com/docs/DOC-47563) technology.

##Dependencies

This sample has a dependency on [Developer-Autodesk/library--dotnet](../library--dotnet). You will also need access to an SAP database and SAP GWM.
